# `.claude/skills` — каркас для навыков Meta-Guard Architect

В этот каталог можно добавлять небольшие \"skills\" — рецепты/паттерны работы для
агентов, например:
- как классифицировать задачу в Trunk / Branch / Leaf;
- как оформлять Mermaid для gitflow или архитектуры;
- как маппить NX‑таск на конкретные сервисы/файлы через RepoScan;
- как выбирать инструмент (Oracle / Codex / CodeMachine) для паттерна задачи.

Сейчас каталог создаётся как основа; конкретные skills будут добавлены позже
после стабилизации NX‑00 серии и CodeMachine спецификаций.
